PASSMETHODS Returns a list of available AT passmethod functions in

SYNOPSIS

function files = passmethods(varargin);

DESCRIPTION 

PASSMETHODS Returns a list of available AT passmethod functions in
 /simulator/element directory
 LIST = PASSMETHODS('mex') returns list of mex-files (default)
 LIST = PASSMETHODS('m')   returns list of m-files
 LIST = PASSMETHODS('c')   returns list of c-files

  NOTES
  1. For input parameters for collaing a pathmethod, use
     [req opt] = PASSEMETHODE
     e.g
     [req opt] = BndMPoleSymplectic4Pass
                 where req are mandatory field and opt are optional fields

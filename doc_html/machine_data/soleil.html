PURPOSE

SOLEIL Loads SOLEIL lattice

SYNOPSIS

DESCRIPTION

  • atmarker ATMARKER Creates a marker space element

  • atsbend ATSBEND Creates a sector bending magnet element with class 'Bend'
  • atsextupole ATSEXTUPOLE Creates a sextupole element with class 'Sextupole'
This function is called by:
  • generateTrackData give some initial coordinates. Track through sample lattices.
  • testTracking test results of tracking against values previously computed

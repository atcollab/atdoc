ABOUT THE TOOLBOX

Accelerator Toolbox (AT) is a collection of tools to model storage rings and beam transport lines in Matlab. With AT it is possible to:
  • create and manipulate accelerator lattice elements, which are stored in Matlab structures
  • track particles through the lattice, selecting the appropriate integrator to represent the physics
  • compute accelerator parameters and beam properties, generating new scripts or taking advantage of the existing ones
Examples of things AT enables are: create a lattice structure that links to the integrators, and then to track through the lattice including non-linear elements, analyze the non-linear motion, change settings to affect parameters, compute equilibrium beam sizes with radiation, and so on.

The toolbox can be downloaded here.

CONTENTS

The toolbox is divided into 7 parts, arranged by directory:
  • integrators Passmethods for tracking (integrators).
  • Machine Data Lattice examples.
  • AT Main Main directory of AT.
  • Demos A few demos and examples.
  • GUIs Graphical interfaces (interactive lattice editor).
  • Matching Matching routines.
  • Physics All the physics related functions.
  • Plotting Versatil atplot function.
  • Tracking Tracking routines (ringpass, linepass, atpass).
  • Lattice tools Element creators, survey, lattice convertors, lattice manipulation functions.
  • Sharing development Tools shared by the AT community.

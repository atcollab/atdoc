atupdateContents

PURPOSE

Updates Contents.m of AT directories

SYNOPSIS

This is a script file.

DESCRIPTION 

 Updates Contents.m of AT directories
 For ATROOT: Contents will include subdirectories
 For other subdirectories, policiy is defined in case by case

CROSS-REFERENCE INFORMATION

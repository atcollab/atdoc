atmexall

PURPOSE

ATMEXALL builds all AT platform dependent mex-files from C-sources

SYNOPSIS

function atmexall(varargin)

DESCRIPTION 

ATMEXALL builds all AT platform dependent mex-files from C-sources

 On UNIX platform, the GNU gcc compiler must be installed and
 properly configured.
 On Windows, Microsoft Visual C++ is required

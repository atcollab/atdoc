function SetESRFAlgeAlignmentError(...
     ring,...          lattice AT
     algeexcelfile,... excel file with sheets: Coords, dR, dL, dZ
     filenameout,...   output filename = ['ALGEErrorsLattice_' num2str(ierr) '_' filenameout]
     nseeds,...        vector of seeds to use

 sets alignment errors defined in excel file algeexcelfile. BPM move with
 the lattice

see also: setDSerr setshift_THERING

CROSS-REFERENCE INFORMATION

This function calls:
  • findspos FINDSPOS returns longitudinal positions of accelerator lattice elements.
  • atgetcells ATGETCELLS performs a search on MATLAB cell arrays of structures
  • atsetshift ATSETSHIFT sets the misalignment vectors
This function is called by:
  • SetLargeErrorList SETLARGEERRORLIST Sets given error list
  • testerr

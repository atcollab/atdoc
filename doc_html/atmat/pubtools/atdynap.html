atdynap

PURPOSE

ATDYNAP Computes the dynamic aperture for ring structure

SYNOPSIS

function [xx,zz]=atdynap(ring,nt,dpp,rfrac)

DESCRIPTION 

ATDYNAP Computes the dynamic aperture for ring structure

  [xx,zz] = atdynap(ring,nturns,dpp,rfrac)

  INPUTS
  1. ring  - Lattice strucutre
  2. nt    - Number of turns for tracking
  3. dpp   - Energy offset (default: 0)
  4. rfrac - Resolution of the grid for checking the stability
              as a fraction of the maximum stable amplitude
              (default: 0.02)
  OUPUTS
  1. XX - Horizontal limit of the dynamic aperture (betatron amplitudes in m)
  2. ZZ - Vertical limit of the dynamic aperture (betatron amplitudes in m)

  NOTES
  1. Algorithm in 2 steps
  2. Speed up possible using a alorithm similar to ELEGANT with dichotomy
  3. Improve and add more input parameter for rlist and np at least
  4. Handle DISPLAYFLAG

  EXAMPLES
  1. ring = soleil; % load lattice
     [xx zz] = atdynap(ring, 100); % compute DA
     plot(xx,zz, '*-'); % plot DA

  See also atmomap

CROSS-REFERENCE INFORMATION
